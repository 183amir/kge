Kochol Game Engine(KGE) is an object-orianted hardware accelerated 3D Game engine. Easy to use and powerful engine for cutting edge games.
KGE is open source and free.

Features

* Object-Oriented Design
* Easy to use and flexible
* Dynamic lighting
* Projected planar shadows
* Shadow mapping
* Multitexturing
* Shader 3 support in ASM, HLSL and GLSL
* MS3D mesh, Skeletal animation
* Refraction, Reflection
* Render To Texture with Anti aliasing
* Multi Rendering Targets (MRT)
* Tile based terrain
* Fog
* Particle systems
* Basic GUI
* 3D sound with OpenAL
* Rigid body physics
* Physics dynamics
* Collision Detection
* Triggers
* Joints
